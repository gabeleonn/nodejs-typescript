# Initial Project for Node.js with Typescript

This is just a simple boilerplate for node.js.


## Commands

To install the packages:
```sh
yarn
```

To start developer mode:
```sh
yarn start:dev
```

To start developer mode with Debug (VSCode Only):
```sh
yarn start:debug
```

To build:
```sh
yarn build
```

To start production:
```sh
yarn start
```
